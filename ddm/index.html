高股息红利股 DDM 概率定价分析模型

基于EPS拆解 + 两阶段DDM + 贝叶斯似然比动态更新 + 隐含回报率决策

一、模型架构

阶段一 (Y1-Y5)人工手动预测绝对EPS和派息率，自动计算股息 D_t = EPS_t × Payout_t
阶段二 (Y6+)戈登增长模型(单阶段DDM)，使用永续增长率 g₂ 和折现率 r

二、三大情景

保守情景盈利承压、派息率下降，g₂=0%，r=8%。尾部风险与铁底测试
中性情景稳健经营、分红政策连续，g₂=2%，r=7%。最大概率基准状态
乐观情景量价齐升或资产荒拔估值，g₂=3%，r=5.5%。期权价值

三、贝叶斯似然比更新机制

Step1 铁律在看财报前，先写好判定标准与似然比L赋值(防止事后合理化)
Step2财报公布后，对照标准赋值L (L>1支持, L<1反对, L=1无信息增量)
Step3后验概率自动计算: 后验_i = (L_i × 先验_i) / Σ(L_j × 先验_j)

四、决策规则

隐含回报率E(R) = Σ(后验概率_i × 单情景隐含回报率_i)，单情景 ≈ D₁/P₀ + g₂
买入门槛Hurdle Rate = 资金成本(如房贷/理财) + 风险补偿(2%)
决策规则E(R) > 买入门槛 → 坚定买入; 否则观望或减仓

五、云端同步

数据存储每股独立文件
安全认证Fine-grained PAT，仅授权目标仓库读写，存浏览器 localStorage
自动保存编辑后 3 秒自动同步到云端，状态栏实时显示
版本回溯每次保存产生 Git commit，可在 GitHub 查看完整历史
命名快照关键时点（如财报前）手动保存快照
冲突检测SHA 校验，远端有更新时提示加载最新

金色 = 自动计算关键结果
绿色 = 最终决策输出
