6. 模型说明、口径与本版修复

数学模型

几何布朗运动：

S(t+Δt) = S(t) × exp[(μ − σ²/2)Δt + σ√Δt · X]

其中 μ 是年化期望收益率，σ 是年化波动率，X 是经由相关矩阵 Cholesky 分解得到的相关标准正态变量。

Student-t GBM（肥尾噪声模型）：

S(t+Δt) = S(t) × exp[(μ − σ²/2)Δt + σ√Δt · Tν]
Tν = [X / √(V/ν)] × √[(ν−2)/ν]，V ~ χ²(ν)

ν 是 Student-t 自由度。系统采用多元 Student-t：共同卡方分母使资产在极端行情下也会出现尾部联动；噪声已标准化为方差 1，并在工程上截断至 ±10 个标准差，避免指数变换产生数值溢出。ν 越小尾部越厚，极端单日冲击、最差路径和深尾部风险通常越突出，但具体 VaR/CVaR 还会受到 σ、期限及中心峰度影响；ν 大于 30 时结果接近正态 GBM。由于 Student-t 指数收益的严格理论均值不适合用正态 GBM 的闭式公式解释，此处的 μ 应理解为年化漂移或近似目标收益参数。

Merton 跳跃扩散：

S(t+Δt) = S(t) × exp[(μ − σ²/2 − λκ)Δt + σ√Δt · X + ΣJ]
κ = exp(μJ + σJ²/2) − 1，μdrift = μ − λκ

跳跃次数服从泊松分布，单次对数跳跃服从正态分布。补偿项 −λκ 用于避免“引入跳跃”本身额外抬高或压低期望收益。系统性跳跃占比用于刻画市场共同冲击；这是比“所有资产跳跃完全独立”更现实的近似，但仍不是完整的跳跃相关结构模型。

Student-t + 跳跃扩散（组合模型）：

S(t+Δt) = S(t) × exp[(μ − σ²/2 − λκ)Δt + σ√Δt · Zν + ΣJ]

同时启用 Student-t 肥尾噪声（方差归一化 Zν）与 Merton 跳跃漂移补偿（−λκ）。两种校准独立运作：Student-t 缩放因子 √((ν−2)/ν) 确保扩散项冲击具备单位方差，使 σ 保持 GBM 中的金融含义；跳跃补偿 κ = exp(μJ + σJ²/2) − 1 修正漂移，确保引入跳跃后期望收益不发生无意识偏离。适用于既需刻画连续肥尾联动、又需刻画不连续跳空的复合尾部风险场景。

组合口径

  • 买入并持有：初始按目标权重买入固定份额，之后不再交易，资产权重会随价格涨跌自然漂移。
  • 每日再平衡：每个交易日收盘后将组合恢复至目标权重。该口径不含交易成本、税费与冲击成本。
  • 系统把初始组合净值统一设为 100。初始价格用于确定持有份额；在归一化展示下，价格本身的比例尺不影响净值结果。

风险指标定义

  • VaR：按期末净值下分位数计算的本金损失，并截断为不小于 0。例如 95% VaR = 12%，表示约 5% 的路径期末损失超过 12%。
  • CVaR / ES：低于相应 VaR 分位点的尾部路径平均损失，用于衡量“跌破阈值后平均亏多少”。
  • 最大回撤：每条路径从历史峰值到后续低点的最大跌幅；面板展示所有路径的平均最大回撤。

本版关键修复

  1. 新增 Student-t 肥尾噪声模型，并在全部模型参数输入框下方补充含义、取值范围和影响说明。
  2. 将“期末净值”和“收益率”分开展示，修正原版把 109.5 的净值直接标成 109.5% 收益的歧义。
  3. VaR/CVaR 改为标准损失口径并限制为非负，避免分位点盈利时出现负风险值。
  4. 新增可复现随机种子；原版 Math.random() 每次不可复现，不利于审计和回归测试。
  5. 新增半正定检查与最近相关矩阵投影，解决原版“说明允许半正定、代码却只接受正定”的矛盾。
  6. 资产表格改为安全 DOM 构建，消除资产名称注入 HTML 的 XSS 风险。
  7. 仿真迁移到 Web Worker，并加入进度与取消，避免大规模计算冻结浏览器主线程。
  8. 增加输入校验、退化直方图处理、权重归一化和结果 CSV 导出。
  9. 新增 Student-t + 跳跃扩散组合模型，同时启用方差归一化与漂移补偿双校准。
  10. 新增模型校准参数面板，实时显示 ν、归一化因子、κ、λκ 及各资产有效漂移 μ_eff。
  11. 引入对偶变差（Antithetic Variates）方差缩减技术，相同路径数下均值估计精度翻倍。
  12. 增加 NaN/Infinity 防护：极端参数下 exp() 钳制与 calculateMetrics 非有限值过滤，避免 UI 显示 NaN。
  13. 修复 Worker Blob URL 内存泄漏：正常完成后释放 objectURL。
  14. Chart.js CDN 添加 SRI 完整性校验，防止 CDN 篡改风险。

重要局限

  • 模型未包含股息、做空约束、融资利率、交易成本、税费、流动性冲击和保证金规则。
  • GBM 的收益率分布偏薄；Student-t 能刻画连续肥尾，Merton 跳跃能产生峰度与不连续路径，但二者都不能替代基于历史数据的压力测试。
  • 预期收益、波动率和相关系数均由用户主观输入，输出对参数高度敏感；不应把模拟结果视为收益承诺或唯一投资依据。
